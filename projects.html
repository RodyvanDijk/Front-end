My projects


Mijn eerste project

Dit project kreeg de naam 'Het ouderavond project', omdat de bodoeling van het project was om een website te bouwen met html en css die uitleg geeft aan de ouders over de school, de opleidingen etc. We moesten ook een form maken maar die mocht niet verwerkt worden. En alles moest gevalideert zijn. Mijn eerste project verliep goed, although ik het moeilijk vond om soms te communiceren met mijn duo. Het project valt hier te vinden.


Mijn tweede project

Mijn tweede project was heel erg gefocused op php en sql, we moesten een website bouwen met login en products in een database en includes en functions etc. Dus ook login sites, een aparte admin login/ klant login. We hadden ook zoek functies gemaakt en verschillende profile pages gemaakt. De website valt hier te bekijken/te bezoeken.


Mijn huidige projecten

Ik ben nu aan het werk aan dit project waarin ik deze site aan het bouwen ben om te laten zien wat ik al heb gedaan en wie ik ben. Deze site is heel erg gefocused om de front-end, en daarom ben ik ook aan het testen met verschillende thema's en andere opties. Ik ben ook aan het werk met een javascript opdracht, ik moet een javascript game bouwen met verschillende levels, die ik dan moet laten zien op een simple website. Op die simpele website zetten ik en mijn werk partner (waarmee ik de opleiding versneld doe) allebei onze eigen game, dit is trouwens niet dezelfde partner uit het eerste project, wel de tweede en javascript.
