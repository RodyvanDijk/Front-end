Mijn learning process


Het begin

Voordat ik begin met het vertellen over mijn website waarin ik leerde over html en css wil ik in het kort even vertellen over het begin van dit hele avontuur. Het avontuur begon gelijk de eerste dag van de eerste schoolweek. We werden geintroduceerd aan html en css! We kregen een paar video's van LittleWebHut. Hier werden de basics uitgelegt. Nu je de basics weet laten we gaat kijken naar mijn websites.


Mijn eerste website

Ik begon natuurlijk met het bouwen van html, gewoon een simpel skelet voor mijn website ontwerpen, dat was deze website. Zoals je kan zien is het niet heel speciaal, maar wel een begin voor iets geweldigs.


Mijn tweede website

In deze website begon ik de basics van html aardig goed te begrijpen en begon ik ook geluiden, video's, fotos en kleuren toe te voegen. Deze website kunt u hier zien.


Mijn derde website

In mijn derde website begon ik met het gebruiken van een html en een css file. Daarom is er ook zo'n groot verschil tussen de vorige websites en deze website. Ik begon te experimenteren met links en backgrounds etc. Deze website die je hier kunt bezoeken.


Mijn vierde website

Deze website was gefocus op wat dingen zelf uitproberen en te testen, hier in maak ik gebruik van een aside etc. Ik heb ook wat rainbow tekst gemaakt. Toen ik deze website af had wist ik dat ik voorbereid was voor het eind project.


Wat toen?

Toen begon ik met het leren van php en ben ik ook meer ingegaan op het goeie gebruik van html en css en volgens de regels erin werken. Op de pagina 'projects' vind je meer informatie over mijn eerste paar projecten en wat ik allemaal van php gedaan heb en wat ik van projecten nog aan werk.
